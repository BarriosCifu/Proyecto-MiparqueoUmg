README - MiparqueoUmg

Instrucciones básicas del proyecto:

1. Este es un proyecto Java Maven para la gestión de parqueos.
2. El código fuente se encuentra en: src/main/java/com/mycompany/miparqueoumg/
3. Para compilar el proyecto, ejecutar:
   mvn clean package
4. El archivo JAR generado se encuentra en: target/MiparqueoUmg-1.0-SNAPSHOT.jar
5. Los recibos generados se almacenan en la carpeta: recibos/
6. Asegúrate de tener Java y Maven instalados para ejecutar el proyecto.
